Arduino Music Player Project Documentation

Introduction:

The Arduino Music Player project is an enjoyable DIY project that allows you to create your own music player using Arduino. You can build a music player that can play ".wav" audio files. This project involves using an Arduino Nano, an SD card module, speakers, and a few basic tools. The Arduino code for this project utilizes the "TMRpcm" library to play audio files stored on a microSD card. This documentation will guide you systematically through the project.

Materials & Supplies:

Parts List:

  • Arduino Nano
  • SD Card Module
  • Speakers

Tools:

  • Soldering Iron
  • Solder Wire
  • Helping Hands

Arduino Library: Install the "TMRpcm" library in the Arduino IDE by going to Tools > Manage Libraries and searching for "TMRpcm."

Implementation:

1. Soldering Components:

• Solder all the components as shown in the circuit diagram.
• The SD card module interfaces with the Arduino using the SPI protocol.

2. Arduino Processing:

• The Arduino reads information from the SD card and generates audio signals.
• These signals are then outputted through the speaker, which is connected to digital pin 9 on the Arduino Nano or Uno.
• For Arduino Mega, use digital pins 5, 6, 11, or 46.

3. Converting Music Files:

• To ensure compatibility with Arduino, convert your music files to the following specifications:
• Samples Per Second (Hz): 16000
• Channel: Mono
• Bits Per Sample: 8
• PCM Format: PCM unsigned 8-bit
• Use the website audio.online-convert.com to convert your music files.
• After conversion, rename the songs as song1, song2, song3, etc.

4. Prepare the microSD Card:

• Format the microSD card as FAT using formatting software like "SD Memory Card Formatter."
• Copy all the converted WAV audio files to the microSD card.

5. Usage:

• Upon resetting the Arduino, the first song (song1.wav) will play automatically.
• Utilize the following buttons for control:
• Play / Pause Button: Play or pause the current track
• Next Button: Play the next track.
• Prev Button: Play the previous track.
